### Scripty Shader Re-coded Edition
  **shader Developed to be compatible for low end PC & mobile (Pojav, Zalith, Etc,,, & any PC)**

> Compatible with Iris, Optifine & Sodium. Mc 1.20 - Mc 26.1.1.

This version will strictly prioritize:

- Compatibility and stability: will confidently say that it will work on all mc versions I've mentioned, unlike the "old and broken" seus code base on my old SSRE shaders where "shader compatibility depend on your hardware capacity".

- Performance: Because I've actually started from "O" and not using the heavy seus code bases from those broken, and super unstable " seus for pojavlauncher " port, this version is also a big change in performance.

- Great visual, Great FPS: Aiming specifically for performance gain doesn't always mean degrading the shader visuals entirely. SSRE shader will and will always still offer those signature vibrant but realistic color cycle and visuals.

- Optimized for low-end Mobile & PC: 90% of codes in this shader used pure math and avoid harmful calcution technique (if-statements, loops, composite, high sample).
